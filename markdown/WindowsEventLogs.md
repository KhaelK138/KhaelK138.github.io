## Basics

- Windows Event Logs cover application errors, security events, diagnostics, service information, etc.
- They're categorized into Application/System/Security and others
  - This also can include "Forwarded Logs", allowing one central computer to have access to logs from multiple others
  - Logs can be saved and opened later/elsewhere as `.evtx` files

**Anatomy**
- Application logs will either be of an Information or Error type
